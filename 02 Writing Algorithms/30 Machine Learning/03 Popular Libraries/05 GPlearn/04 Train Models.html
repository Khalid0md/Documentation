In this example, a genetic programming feature transformation model and a genetic programming regression prediction model using the following features and labels will be trained:

Data Category Description
Features Daily percent change of the close price of the SPY over the last 5 days
Labels Daily percent return of the SPY over the next day

Follow these steps to create a method to train your model:

History Call

You need historical data to train the model. Call History with Symbol, timerule, and resolution to obtain historical data. In this example, 2 years of daily trade bar data is used.

self.History(self.spy, 252*2, Resolution.Daily)

Prepare Data

Follow the below steps to create a method to prepare the data for training and prediction.

  1. Create a method to process the data for the algorithm class, with input data and number of timestep as arguments.
    2. def ProcessData(self, history, n_steps=5):
  2. Call the pct_change and dropna methods.
    3.     daily_returns = history['close'].pct_change().dropna()
  3. Loop through the daily_returns DataFrame and collect the features and labels.
    4.     features = []
    labels = []
    for i in range(len(daily_returns)-n_steps):
        features.append(daily_returns.iloc[i:i+n_steps].values.flatten())
        labels.append(daily_returns['close'].iloc[i+n_steps])
  4. Convert the lists of features and labels into numpy arrays and return them.
    5.     features = np.array(features)
    labels = np.array(labels)

    return features, labels

Build Model

Follow the below steps to build the model.

  1. Get processed training data.
    2. features, labels = self.ProcessData(history)
  2. Declare a set of functions to use for feature engineering.
    3. function_set = ['add', 'sub', 'mul', 'div',
                'sqrt', 'log', 'abs', 'neg', 'inv',
                'max', 'min']
  3. Call the SymbolicTransformer constructor with the preceding set of functions. Save it as a class variable.
    4. self.gp_transformer = SymbolicTransformer(function_set=function_set)
  4. Call the SymbolicRegressor constructor to instantiate the regression model.
    5. self.model = SymbolicRegressor()

Fit Model

Follow the below steps to fit the model with the prepared data.

  1. Get processed training data.
    2. features, labels = self.ProcessData(history)
  2. Call the fit method of the SymbolicTransformer with the training features and labels.
    3. self.gp_transformer.fit(features, labels)
  3. Call the transform method with the training features to transform the data.
    4. gp_features = self.gp_transformer.transform(features)
  4. Call the hstack method with the original features and transformed features to stack them column-wise.
    5. new_features = np.hstack((features, gp_features))
  5. Call the fit method of the SymbolicRegressor with the stacked features and labels.
    6. self.model.fit(new_features, labels)
