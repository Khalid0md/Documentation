You must import the plotting libraries and get some historical data to create scatter plots.

In this example, you create a scatter plot that shows the relationship between the daily price of SPY & TLT. Follow these steps to create the scatter plot:

  1. Get historical data for the 2 securites.
    2. var history = qb.History<TradeBar>(new[] {spy, tlt}, new DateTime(2022, 1, 1), new DateTime(2023, 1, 1), Resolution.Daily);
  2. Create a Line chart.
    3. var chart = Chart2D.Chart.Point<decimal, decimal, string>(
    history.Select(x => x[spy].Close),
    history.Select(x => x[tlt].Close)
);
  3. Create a Layout.
    4. LinearAxis xAxis = new LinearAxis();
xAxis.SetValue("title", "SPY Price ($)");
LinearAxis yAxis = new LinearAxis();
yAxis.SetValue("title", "TLT Price ($)");
Title title = Title.init($"SPY vs TLT");

Layout layout = new Layout();
layout.SetValue("xaxis", xAxis);
layout.SetValue("yaxis", yAxis);
layout.SetValue("title", title);
  4. Assign the Layout to the chart.
    5. chart.WithLayout(layout);
  5. Show the plot.
    6. HTML(GenericChart.toChartHTML(chart))

    The Jupyter Notebook displays the scatter plot.
