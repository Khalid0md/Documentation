We recommend Visual Studio Code Dev Containers for dataset development and testing. This extension lets you use a Docker container as a full-featured development environment.

Follow these set up your dockerized environment:

  1. Install Visual Studio Code.
    1. Get Dev Containers extension.
  2. Install Docker.
    1. Follow the instructions for your Operating System
    2. New to Docker? Try docker getting-started.
  3. Pull Lean's latest research image from a terminal:
    4. $ docker pull quantconnect/research:latest
